b标签能加粗字体，i标签能把文字变成斜体，u标签给文字加下划线

img标签可以插入图片，src属性里写图片地址，还可以指定图片大小（px是一种计算机图像长度单位，意思是像素）

a标签添加超链接，href里写完整链接，这个链接会在当前窗口打开 芒果TV

新窗口打开链接要写一个target参数 github

div标签给段落分块，一行只能放一个div

span给一行分块，一行能放好几个span（分块有助于段落更清晰）
